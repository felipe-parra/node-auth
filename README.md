# Node-Auth

_Clean Architecture_

## Getting Started

1. Clone repo

```bash
git clone https://github.com/felipe-parra/node-auth.git
```

2. Move to directory

```bash
cd node-auth
```

3. Install packages

```bash
pnpm install
```

4. Add your environment variables

```bash
PORT=3000
```

5. Run project on development

```bash
pnpm run dev
```
